#The-Tech-Academy-Basic-C-Sharp-Projects

These are some of the C# projects I created while learning C#, .NET Framework, and Entity Framework (Code First).
Each project helped me strengthen my understanding of programming logic, database connections, and real-world aa-design.

##My Projects

- [Car Insurance](https://github.com/kraken-wannabe/The-Tech-Academy-Basic-C-Sharp-Projects/tree/main/CarInsurance)
- [Code First Student](https://github.com/kraken-wannabe/The-Tech-Academy-Basic-C-Sharp-Projects/tree/main/CodeFirstStudent)

---

### Car Insurance
An MVC web app that calculates car insurance quotes based on user input.
It includes an admin page where staff can view and manage all the submitted quotes.

--Highlights--
- Developed using ASP.NET MVC
- Connected to a SQL database
- Custom quote calculated logic
- Admin dashboard for managing records

-----------

### Code First Student
A console app that demonstrates Entity Framework (Code First).
It automatically creates a SQL database and performs basic CRUD operations to manage student data.

--Highlights--
- Uses Entity Framework 6
- Code First database creation and migrations
- LINQ used for handling and displaying data
